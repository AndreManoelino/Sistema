Criando aplicação para agendamentos em diversos salao de beleza.

Codificando com o codeigniter 4 utilizando boas práticas de desenvolvimento com camada mvc

Utilização do banco de dados Postgres para armazenamento desses dados. Utilização do PHP 8.

Inicialmente foi criado a estrutura inicial de um menu para nos desenvolvedores onde criamos o salão para o cliente e temos controle com as urls para dono e para cliente do salão.
